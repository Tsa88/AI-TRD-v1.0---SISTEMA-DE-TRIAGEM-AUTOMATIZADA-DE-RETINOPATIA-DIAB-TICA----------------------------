# Contribution Guidelines

Thank you for your interest in contributing to the AI-TRD project! We appreciate your help in improving our project.

## How to Contribute

1. **Fork the Repository:** Click on the fork button at the top right corner of the screen.
2. **Clone your Fork:** Clone your copy of the repository to your local machine:
   ```bash
   git clone https://github.com/Tsa88/AI-TRD-Triagem-Automatizada-de-Retinopatia-Diab-tica-.git
   ```
3. **Create a Branch:** Create a new branch for your feature or bug fix:
   ```bash
   git checkout -b my-feature-branch
   ```
4. **Make your Changes:** Implement your changes or add features in this new branch.
5. **Commit your Changes:** Commit your changes with a clear message:
   ```bash
   git commit -m "Add my new feature"
   ```
6. **Push Changes:** Push your feature branch to your GitHub fork:
   ```bash
   git push origin my-feature-branch
   ```
7. **Create a Pull Request:** Go back to the original repository, and you'll see an option to create a pull request. Follow the instructions to do so.

## Development Setup Instructions

To run this project locally, follow these instructions:

1. **Clone the Repository:** If you haven’t already, clone the repository using the command mentioned above.
2. **Install Dependencies:** Navigate to the project directory and install the necessary dependencies:
   ```bash
   npm install
   ```
3. **Run the Project:** Start the development server:
   ```bash
   npm start
   ```
4. **Access the Application:** Open your browser and go to `http://localhost:3000` to view your application.
